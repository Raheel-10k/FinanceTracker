# AI Cashflow Survival Copilot

A premium, CRED-inspired, mobile-first web application that analyzes cashflow statements intelligently.

## Project Structure
- `frontend/`: React + Vite + TypeScript + Tailwind CSS application. Use this for the UI.
- `backend/`: Node.js + Express backend skeleton built and configured for MongoDB and PDF/CSV parsing.

## How to run locally

### Frontend (UI Prototype)
The frontend UI is completely implemented with the dark premium design system.
```bash
cd frontend
npm install
npm run dev
```
It will start on `http://localhost:3000`. We recommend viewing it in the Google Chrome developer tools toggled strictly to mobile view (e.g. iPhone 14 Pro), or loading it via your phone.

### Backend (API Engine)
The backend requires setting up MongoDB. Check `package.json` for all required scripts and modules. 
```bash
cd backend
npm install
npm run dev
```

Enjoy your premium fintech app!
